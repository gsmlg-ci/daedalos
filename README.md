# daedalos

[![Build](https://github.com/gsmlg-ci/daedalos/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/daedalos/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/daedalos`
- `ghcr.io/gsmlg-dev/daedalos`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/daedalos:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/daedalos:latest
```

## Build

```bash
docker build -t gsmlg/daedalos .
```
